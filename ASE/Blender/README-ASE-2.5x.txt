Installation
============
Custom scripts need to be placed in the following location

- scripts\io\

For Windows Vista/Win7 users this is typically located in

- C:\Users\[profile]\AppData\Roaming\Blender Foundation\Blender\2.54\scripts\io

XP users should look in 

- C:\Documents and Settings\[profile]\Application Data\Blender Foundation\Blender\2.54\scripts\io
